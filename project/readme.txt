Участники: Комарова О. Кондратьева А. Нужденко И.

Идея: визуализировать связи между группами вк по подписчикам группы Банка СПБ

Методы визуализации:
- граф для отображения связей между группами
- при клике на вершину отобржаются связанные с выбранной группы, открывается информация по группе и список схожих групп 
- таблица ТОП 10 +сортировка по клике на шапку
- фильтр по году (год даты первого поста)
